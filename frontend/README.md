# T³ Online

Next.js app router frontend for the Tic Tac Toe AI project.

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
pnpm lint
```

## Notes

- Routes live under `src/app`.
- Shared UI and game components remain in `src/components`.
- Theme state is persisted through Zustand and applied via body classes.
